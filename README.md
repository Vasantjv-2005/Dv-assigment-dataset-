##  Overview

This project focuses on analyzing a student performance dataset and creating meaningful visualizations using Tableau. The goal is to understand patterns, comparisons, and relationships in the data through different types of charts.

---

## Dataset Description

The dataset used in this project contains information about students, including:

* Gender
* Internet Access
* Study Hours Per Week
* Final Score
* Attendance Rate
* Parent Education

The dataset consists of approximately **60–80 records**, fulfilling the assignment requirement.
